Spring、JDK、Maven 版本兼容性表格

2025-01-27 4 分钟阅读 javaspringboot

升级概述

  • JDK 升级到 21 LTS（因为是当前支持周期最长的版本）
  • Maven 升级到 3.9.9
  • Spring Boot 升级到 3.4.1（因为是当前的最新版）
